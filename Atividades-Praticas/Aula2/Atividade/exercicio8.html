Tags HTML


  1. TAG: !DOCTYPE html: Declaração do tipo de documento. Indica que o documento é uma página HTML5.


  2. TAG: p: é usada para definir parágrafos de texto.


  3. TAG: a: é usada para criar links. Atributos como href permitem definir o destino do link


  4. TAG: header Representa o cabeçalho da página, geralmente contendo títulos, logotipos e menus de navegação.


  5. TAG: main Contém o conteúdo principal da página. Não deve ser usado para elementos repetitivos, como o cabeçalho, rodapé ou barras de navegação, que têm um propósito secundário. O conteúdo dentro da tag main é único e relevante para o tema principal da página.


  6. TAG: li Representa um item de lista. Cada li dentro de um ol é numerado automaticamente na ordem em que aparece.


  7. TAG: ol Define que a lista é ordenada. Os itens dentro dela são automaticamente numerados pelo navegador.


  8. TAG: footer: Representa o rodapé da página, geralmente contendo informações de copyright, links de contato e outras informações relevantes.
