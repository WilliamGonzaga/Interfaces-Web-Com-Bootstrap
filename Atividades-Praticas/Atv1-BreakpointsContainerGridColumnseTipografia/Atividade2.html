Importação do Bootstrap via CDN

O uso de CDN para importar o Bootstrap oferece várias vantagens, como a redução do tempo de carregamento, já que os arquivos são servidos a partir de servidores rápidos e distribuídos globalmente. Além disso, como o arquivo pode já estar em cache do usuário (caso ele tenha visitado outro site que usa o mesmo CDN), o desempenho é otimizado. Outro benefício é que a manutenção e atualização dos arquivos são feitas automaticamente, sem necessidade de intervenção do desenvolvedor. Uma desvantagem do uso de CDN é a dependência da conexão com a internet. Se o usuário tiver problemas de conexão ou se o CDN estiver temporariamente fora do ar, o site pode ficar inacessível. Além disso, embora o arquivo seja servido rapidamente, se o usuário estiver em uma rede com alta latência ou se o CDN não for bem distribuído na sua região, pode haver um impacto no desempenho.
